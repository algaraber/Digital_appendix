This directory contains:

- BRLC-lite.zip
- sample_data.zip

These are encrypted zip files containing the working version of the classifier script described in chapter 3 of the thesis*, including the minimap indexes, and a sample dataset that allows the script to be tested. The sample dataset is the one obtained in section 3.2.3.1 of the thesis.

Thesis refers to Alvaro Garcia Delgado's "Molecular approaches for pathogen assessment in remote aquaculture settings"; the first version submitted to Aberystwyth University (October 2021)
